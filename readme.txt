Enables you to integrate Google Analytics in your site.

== Description ==

Google Analytics allows you to analyse how your visitors find you, what they do on your site, how long they’ll stay on particular page, where you lose them and much more.   

== Installation ==

1. Upload 'smntcs-google-analytics’ to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to https://www.google.com/analytics/, add a new site and copy the site verification code of the meta tag
3. Click on the new menu item "Settings" » “Google Analytics“ and enter your site verification code
